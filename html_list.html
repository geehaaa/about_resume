We promote the basic design concepts:

  1. Contras
    • Size, color, weight, opacity, etc
    • Create a hierarchy of importance
  2. Repitition
    • Repeat similar elements (color,shape,size,etc) to make the design cohesive and balanced
  3. Alignment
    • All the elements should have a locational relationship to each other (lined-up)
    • Direct the flow and direction of the users's focus
  4. Proximity
    • Group similar components for clearer definitions "Blocks" of ideas make a more intersting layout and will more likely hold the users attention
